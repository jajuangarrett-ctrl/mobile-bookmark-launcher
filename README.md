# Mobile Bookmark Launcher

Touch-first Obsidian launcher for Franklin's mobile vault workflow.

## Features

- Recent Notes tab tracked independently by this plugin.
- Bookmarks tab for Obsidian core bookmarks, including bookmark groups.
- Mobile-sized tap targets and no keyboard shortcut dependency.

## Development

```bash
npm install
npm run build
npm test
```

Copy `manifest.json`, `main.js`, and `styles.css` into:

```text
/Users/franklingarrett/FJG Vault/.obsidian/plugins/mobile-bookmark-launcher/
```
